## Counting Time Complexity

```c#
for(i=0; i<n; i++) --> O(n)
for(i=0; i<n; i=i+2) --> n/2 O(n)
for(i=0; i<n; i--) --> O(n)

for(i=0; i<n; i=i*2) --> O(log2n)
for(i=0; i<n; i=i*3) --> O(log3n)

for(i=0; i>n; i=i/2) --> O(log2n)

```
